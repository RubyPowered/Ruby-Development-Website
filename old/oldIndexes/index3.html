What is Ruby Development?

Well first a common misconception: People for some reason see the name Ruby and think all of this is written in Ruby, it's NOT. All of my programs are written in pure Java.

Ruby Development was founded sometime in early 2020 when I started learning Java and making apps with it for Windows, and they took on the Ruby Development branding later in 2020-early 2021. I have started to make more complex applications for Windows, Mac, and most Linux distros excluding Ruby Launcher that will have a seperate version for all OSs.
