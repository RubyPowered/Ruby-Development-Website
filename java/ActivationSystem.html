Activation System by Ruby Development



Ruby Development now has an activation system! The Ruby Development Activation System is written in pure Java 16 and is a great way to keep software pirates away. It works by taking a username and running it through an encryption algorithm that generates a series of random characters. Then it looks at the entered key and if they match it activates, but if they don't it gives the user a message saying they need to buy a license.
